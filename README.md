Ron's Retro Stacker - RonDevHub Edition
</h1>

Eine hochperformante, ressourcenschonende **Brick-Stacker** im modernen Retro-Look. Entwickelt als PHP-basiertes Projekt, optimiert für mobilfreundliches Gameplay.

## 🚀 Features
* **Drei Retro-Themes:** Wähle zwischen *Gameboy*, *Neon* und *Dark Mode* (deine Auswahl wird gespeichert!).
* **Pro-Vorschau:** Behalte die nächsten drei Blöcke in der Queue immer im Blick.
* **DAS (Delayed Auto Shift):** Schnelle Blockbewegung bei gedrückter Taste – wie bei den Profis.
* **Responsive Design:** Optimiert für Desktop und mobile Endgeräte (Touch-Support ohne lästiges Markieren).
* **Persistent Highscore:** Dein Rekord bleibt lokal im Browser gespeichert.
* **Ressourcenschonend:** Minimaler Fußabdruck, perfekt für kleine Homelabs (getestet auf 12GB RAM Systemen).

## 🛠️ Tech-Stack
* **Backend:** PHP (Pure)
* **Frontend:** Vanilla JS, HTML5 Canvas, CSS3

## 🎮 Steuerung
| Aktion | Tastatur | Mobile Steuerung |
| :--- | :--- | :--- |
| **Links bewegen** | Pfeiltaste Links | ◀ Button |
| **Rechts bewegen** | Pfeiltaste Rechts | ▶ Button |
| **Block drehen** | Pfeiltaste Oben | ↻ Button |
| **Soft Drop** | Pfeiltaste Unten | ▼ Button |
| **Pause** | Taste P | Pause-Button |
